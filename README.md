## Setup

### Backend

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py migrate
```

```bash
python manage.py runserver
```

### Frontend

```bash
cd frontend
```

```bash
npm install
```

```bash
npm start
```
